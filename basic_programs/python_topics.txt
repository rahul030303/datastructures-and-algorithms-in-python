#4. In Python, break and continue statements can alter the flow of a normal loop.
The break statement is used to skip the rest of the code inside a loop for all iterations.(skips the complete loop).
The continue statement is used to skip the rest of the code inside a loop for the current iteration only.
Loop does not terminate but continues on with the next iteration.
